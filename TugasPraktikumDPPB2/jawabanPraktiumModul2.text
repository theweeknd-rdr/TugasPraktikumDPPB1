Soal Praktikum

1. List dengan Null

(a) Membuat list menggunakan List.filled() untuk menentukan ukuran awal.

(b) Menggunakan final dan mengisi nama serta NIM di indeks tertentu.

2. Set (Data Unik)

(a) Deklarasi Set untuk menyimpan data unik.

(b–c) Gunakan tipe eksplisit untuk Set kosong agar tidak error.

(d) Tambah data dengan .add() dan .addAll().

3. Map (Key-Value)

(a) Buat Map dengan berbagai tipe key.

(b–c) Gunakan constructor Map dan isi data secara manual.

(d) Tambahkan atribut seperti asal sekolah dan asal kota.

4. Spread Operator

(a) Gunakan ... untuk menggabungkan dua list.

(b–c) Gunakan ...? agar tetap aman jika salah satu list bernilai null.

(d) Buat list berisi digit NIM secara terpisah.

5. Queue (FIFO)

(a) Deklarasikan Queue menggunakan Queue().

(b–c) Lakukan operasi add() dan remove() sesuai prinsip FIFO.

(d) Masukkan NIM ke Queue dan keluarkan elemen satu per satu dengan while.

6. String Cantik "HEI ⭐"
Langkah algoritma:

Gunakan kombinasi Queue, List, dan Map sesuai kebutuhan.

Temukan semua posisi huruf H, E, dan I pada string.

Gunakan algoritma greedy untuk membentuk pola “HEI” sebanyak mungkin.

Hasil akhir berupa panjang = jumlah pola × 3. Jika tidak ditemukan pola “HEI”, hasil = -1.
